# Phantty v0.22.1

## Fixes

- Fixed Agent skill discovery so `/skills`, `$skill-name` preloading, and the
  `skill_info` tool can find skills from configured roots instead of only
  `./skills`.
- Added support for portable Agent skills beside `phantty.exe`, allowing skills
  shipped with the release package to load regardless of the process working
  directory.

## Packaging

- Installs the eval skill fixtures into `zig-out/bin/skills` during `zig build`
  so the default PDF and browser skills are available next to the executable.
- Copies packaged skills into both portable release zip layouts and validates
  their presence in the Windows release workflow.
