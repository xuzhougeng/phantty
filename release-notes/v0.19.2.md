# Phantty v0.19.2

## Highlights

- Reduced terminal-core memory use by sizing renderer cell and snapshot buffers
  to the active grid instead of reserving a large fixed buffer per surface.
- Added a `-Dwebview=false` build option so pure terminal-core builds can skip
  the WebView2 bridge entirely for memory attribution tests.

## Fixes

- Closing the Browser panel now destroys the embedded WebView2 instance instead
  of only hiding it, allowing the WebView2 runtime and child processes to exit.
- Removed an idle per-surface renderer helper thread that was not yet doing
  snapshot or rebuild work.

## Diagnostics

- Added `phantty-debug-memory` logs for attributing memory across terminal pages,
  renderer buffers, font atlas/cache state, Kitty images, and process counters.
