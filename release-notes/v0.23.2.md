# Phantty v0.23.2

## Fixes

- Fixed URL clicks on builds without a usable embedded WebView2 runtime so `Ctrl`-clicking `http://` and `https://` links opens the system default browser instead of showing an empty right-side browser panel.

## Packaging

- Added `phantty-windows-portable-no-webview-vX.Y.Z.zip`, built with embedded WebView2 disabled.
- Updated the Windows release workflow to validate, upload, and publish the no-WebView portable package alongside the existing portable and WebView2 packages.
