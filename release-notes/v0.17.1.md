# Phantty v0.17.1

## Fixes

- Fixed a minimize/restore regression where activating Phantty from the
  taskbar could shrink the left tab sidebar to its minimum width. Minimized
  `WM_SIZE` events are now ignored for layout sizing, and transient mouse
  drag/press state is cleared while minimized.
- Hardened zero-sized and very small layout paths so split rectangles and the
  main render loop do not continue with invalid framebuffer dimensions.
- Fixed copied multiline shell commands gaining trailing spaces from the split
  pane width. Selection copy now trims trailing clipboard spaces per row while
  preserving leading indentation, matching Ghostty's default behavior.
- Hardened Markdown/text preview rendering against invalid or truncated UTF-8
  so Ctrl-click preview cannot crash while rendering file names or preview
  panel text.
