# Phantty v0.22.2

## Improvements

- Added a Stop control to Phantty Remote AI Chat so remote sessions can cancel
  an in-flight local agent request.
- Added a `^V` shortcut key to the Phantty Remote mobile virtual keyboard.
- Added small-file drag-and-drop upload for SSH terminal tabs. Dropping a local
  file on an SSH terminal uploads it to the current remote directory and inserts
  the uploaded remote path into the input line.
- Improved Agent terminal context handling with explicit terminal selection so
  commands run in the intended panel while still allowing agents to switch
  between multiple panels or servers.

## Fixes

- Hardened the remote relay against malformed websocket text frames and invalid
  UTF-8 in desktop layout JSON.
- Kept Remote AI Chat Stop compatible with older desktop layout payloads that
  do not yet include request runtime state.
