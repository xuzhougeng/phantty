# Phantty v0.22.0

## Improvements

- Added command-center Agent entrypoints:
  - `New Agent` now opens a new Agent tab directly with the default AI profile
  - `Select Agent History` opens a command-center-native picker of saved agent
    conversations
- Added a command-center history picker that lists persisted agent sessions by
  newest activity first and reopens them by `session_id`.
- Improved command-center state handling for agent flows by separating picker
  state into a pure Zig state module and keeping history-row refresh/cleanup
  explicit.

## Fixes

- Fixed `New Agent` so it no longer depends on the first saved AI profile being
  chat-vs-agent compatible; the command now explicitly forces agent launch
  intent.
- Fixed command-center history picker cleanup and refresh behavior so leaving
  the picker or opening settings no longer keeps stale owned history rows
  alive.

## Documentation

- Updated the public docs site to describe the new command-center `New Agent`
  and `Select Agent History` flows, plus the left-sidebar `AGENT History`
  page.
