# Phantty v0.15.0

This release adds background image support so a wallpaper can show
behind the terminal, with adjustable opacity and four scaling modes.

## Highlights

- Added `background-image`, `background-opacity`, and
  `background-image-mode` config keys (matching CLI flags). PNG, JPG,
  BMP, GIF, and TGA are supported through the bundled stb_image.
- Added four scaling modes: `fill` (default, cover and crop),
  `fit` (letterbox), `center` (1:1 pixel scale), and `tile` (repeat
  at native size).
- `background-opacity = 1.0` keeps the theme background fully opaque
  (image hidden); `0.85` is a faint watermark; `0.0` shows the image
  at full strength. The opacity is applied as a theme-color tint
  blended over the wallpaper, so default cells, selections, and ANSI
  backgrounds all respect the same value.
- The wallpaper is rendered inside the post-process FBO, so custom
  shaders set with `--custom-shader` distort it alongside the terminal
  content for a consistent look.
- Hot-reload (Ctrl+,) swaps the wallpaper without restarting; clearing
  `background-image` removes it.

## Example config

```text
background-image = C:\Users\me\Pictures\wallpaper.png
background-opacity = 0.85
background-image-mode = fill
```

## Fixes and polish

- Reused matching SSH browser tunnel ports when reopening links from
  the same remote endpoint.
